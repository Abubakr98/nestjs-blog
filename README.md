## Installation

```bash
cd api/
```

```bash
$ npm install
```

```bash
cd ui/
```

```bash
$ npm install
```

## Running the app

```bash
# do the following commands in the root folder, by default it is /nest-blog

# docker compose, before this command you should have docker running
$ npm run compose

# api
$ npm run api

# ui
$ npm run ui
```
