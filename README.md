## Installation


```bash
$ git clone https://github.com/Abubakr98/nestjs-blog.git

$ cd nestjs-blog/api/

$ npm install
cd ..

$ cd ui/

$ npm install
$ cd ..
```

## Running the app

```bash
# do the following commands in the root folder, by default it is /nest-blog

# docker compose, before this command you should have docker running
$ npm run compose

# api
$ npm run api

# ui
$ npm run ui
```
